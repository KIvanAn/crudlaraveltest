## Simple CRUD on Laravel 8
### Launch project:

- **install PHP-7.4+**
- **install Composer**
- **install Docker**
- **cd /all/your/projects/are/located**
- **git clone https://github.com/KIvanAn/crudlaraveltest.git project-name**
- **cd project-name**
- **composer install**
- **cp .env.example .env**
- **php artisan key:generate**
- **./vendor/bin/sail up**
- **./vendor/bin/sail artisan migrate --seed**
